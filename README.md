# BJ BAR 告知文ジェネレーター 開発レポート

高円寺の駄菓子BAR「BJ BAR」専用のツイート生成ツールを実装しました。

## 実装内容
- **ファイル名**: [index.html](file:///Users/hoshikawayuuki/Documents/告知文/index.html)
- **主要技術**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
- **フォント**: DotGothic16 (レトロ感), M PLUS Rounded 1c (親しみやすさ)

## デザインのこだわり
- **高円寺レトロ・フューチャー**: 黒を基調にマゼンタ（#ff00ff）とシアン（#00ffff）のグラデーションを使用し、夜のバーの雰囲気を演出。
- **ガラスモーフィズム**: 背面のグラデーションが透ける半透明のカードデザインを採用。
- **マイクロアニメーション**: カード上部の光が流れるラインや、ボタンのホバーエフェクトなどで高級感を演出。

## 機能ハイライト
- **スマート日付設定**: 起動時に自動で「次の土曜日」の日付を選択状態にします。
- **140文字ガード**: 全角140文字を超えないよう自動で文章を調整し、文字数カウンターで視覚的にフィードバック。
- **バリエーション生成**: 
    - 季節の挨拶（月ごと）
    - 天気・雰囲気別の本文（5つのカテゴリー、複数パターン）
    - 来店を促す文言（複数パターン）
    - 末尾の「ホシカワ」
- **ワンクリックアクション**: 生成した文章を即座にコピー、またはX（Twitter）の投稿画面へ飛ばすことができます。

## 使い方
1. [index.html](file:///Users/hoshikawayuuki/Documents/告知文/index.html) をブラウザで開きます。
2. 必要に応じて日付や天候を選択します。
3. 「告知文を生成」ボタンを押します。
4. 内容を確認し、「コピー」または「Xでシェア」をクリックして投稿します。

## スマホ・アプリ化への対応 (PWA)
このツールはスマホでも快適に、且つアプリとして利用できるように以下の設定を追加しました。
-   **PWA対応**: iPhoneやAndroidの「ホーム画面に追加」を行うことで、専用アプリのように起動できます。
-   **iOS最適化**: 入力時に画面が勝手にズームしないよう、フォームのサイズを調整済みです。
-   **オフライン動作**: 一度開けば、電波のない場所でも起動が可能です。

## スマホで使う方法
スマホでこの仕組みを実現するには、以下の手順がおすすめです。

### 1. インターネットに公開する（GitHub Pages）
無料で公開でき、スマホからアクセス可能になります。
1.  このフォルダのファイルをすべて [Hoyu55y/bjbar](https://github.com/Hoyu55y/bjbar) にアップロードします。
2.  リポジトリの **Settings > Pages** から `main` ブランチを公開設定にします。
3.  発行されたURL（`https://hoyu55y.github.io/bjbar/`）にスマホのブラウザでアクセスします。

### 2. ホーム画面にアプリとして追加
-   **iPhone**: SafariでURLを開き、真ん中の「共有（↑）」ボタンから「**ホーム画面に追加**」をタップ。
-   **Android**: ChromeでURLを開き、右上の「︙」ボタンから「**ホーム画面に追加**」または「**アプリをインストール**」をタップ。

これで、ホーム画面に今回生成したアイコンが表示され、いつでもサッと起動できるようになります。
